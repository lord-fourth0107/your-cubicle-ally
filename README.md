# Your Cubicle Ally

Hackathon 2026 !!! Lezzz GOOO

An Electron app for personalized onboarding. Enter your job description and complete tailored training modules.

## Getting Started

```bash
npm install
npm start
```

## App Flow

1. **Welcome** — Loading screen with "Start My Onboarding" button
2. **Job Description** — Enter your role details for personalized content
3. **Trainings List** — Clickable onboarding modules to complete
